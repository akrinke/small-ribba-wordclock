\section{Inbetriebnahme der Uhr}\label{sec_inbetriebnahme}
		
		Die Steuerung der Uhr erfordert eine Infrarot"=Fernbedienung.
		
		Es wird eine Vielzahl von Protokollen unterstützt, 
		so dass eine (beinahe) beliebige, überflüssige Fernbedienung benutzt werden kann. 

		Nach Anschluss der Uhr an die Stromversorgung, blinken die vier Minutenpunkte für einige Sekunden. 
		
		Anschließend wechselt die Uhr in den \normalModeName.
